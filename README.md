# voice-assistant(STT + TTS)
 Overview

This project is a real-time voice assistant built using Python that can:

Convert speech to text (Speech-to-Text, STT)
Process user input using logic or AI
Respond using Text-to-Speech (TTS)

The assistant continuously listens, understands commands, and responds in real time.
Features
 Real-time voice input using microphone
Speech recognition (Google Speech API via speech_recognition)
 Intelligent response handling (rule-based / AI-based)
 Text-to-speech output using pyttsx3
 Continuous interaction loop
 Technologies Used
Python
speech_recognition (STT)
pyttsx3 (TTS)
pyaudio (microphone input)
OpenAI API for AI responses
